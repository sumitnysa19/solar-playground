# Features

## Solar System simulation

- Planetary bodies with orbital motion and axial rotation
- Orbit curves rendered as lines
- Atmosphere and cloud layers where available
- Labels with distance-based visibility
- Camera controls (rotate, zoom, pan) with damping
- Time controls (pause, accelerate, reverse)
- Info panel for selected bodies

## Night sky viewer

- Observer-based sky dome
- Stars and constellations from catalog data
- Sun, Moon, and planets rendered as billboards
- Horizon ring and compass markers
- Optional catalog or dome rendering modes

## Interaction features

- Single-click selection and focus
- Raycast-based picking of bodies and labels
- Location click on Earth updates latitude/longitude
- Horizon disc placed as a tangent plane at the click location

## Visual and rendering features

- Bloom post-processing
- Custom shaders for orbits and atmosphere
- Basis texture loading for planet surfaces
- Sky dome background and starfield
- Ambient audio playback with user-gesture resume
